# AquaHarmony - Cooperative Board Game

A Monopoly-inspired educational game where 3-4 players work together to build sustainable water infrastructure in Pu Ngaol.

## About

Players collaborate to complete three main tasks:
1. Build Water Filtration System
2. Build Water Pump Generator  
3. Develop Software Monitoring System

Manage resources (money, wood, steel, electricity, workforce, sand, time) while navigating disasters on a 40-square board.

## Requirements

- Java 11+
- Maven 3.6+

## How to Run
```bash
mvn compile exec:java -Dexec.mainClass="AquaHarmony.Main"
```

## How to Test
```bash
mvn test
```
